
# Cypress Dropdown Examples

This repository contains examples of how to handle dropdowns in end-to-end testing using Cypress.

## Getting Started

### Prerequisites

Ensure you have Node.js and npm installed.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/gurudattgd04/cypress-dropdown-examples.git
   ```
2. Navigate to the project directory:
   ```bash
   cd cypress-dropdown-examples
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Running the Tests

To run the tests, use the following command:
```bash
npx cypress open
```

## Project Structure

- `cypress/`: Contains test scripts and support files.
- `.gitignore`: Specifies files to ignore in the repository.
- `browserstack.json`: Configuration for BrowserStack integration.
- `cypress.config.js`: Cypress configuration file.
- `package-lock.json` and `package.json`: Manage dependencies.
